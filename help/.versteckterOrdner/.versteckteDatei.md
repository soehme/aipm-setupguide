# Du hast mich gefunden!

Diese Datei liegt in einem **versteckten Ordner** (`.versteckterOrdner`). Dateien und Ordner, die mit einem Punkt `.` beginnen, werden normalerweise nicht angezeigt -- weder im Finder (Mac), im Explorer (Windows) noch in Obsidian.

## Warum versteckte Dateien?

Programme nutzen versteckte Ordner für Konfiguration und interne Daten:

- `.obsidian/` -- Obsidians Einstellungen und Plugins
- `.git/` -- Versionsverlauf eines Projekts
- `.claude/` -- Claude Code Einstellungen

## Warum ist das relevant?

Als Product Manager oder im Umgang mit KI-Tools ist es hilfreich zu wissen, dass diese versteckten Dateien existieren. Manche enthalten wichtige Kontextinformationen (wie `CLAUDE.md`), die das Verhalten von Tools beeinflussen.

Mit dem Plugin **"Show Hidden Files"** in Obsidian kannst du diese Dateien sichtbar machen -- genau so hast du diese Datei hier gefunden.
